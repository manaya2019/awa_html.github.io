Découvre comment faire des visualisations avec Plotly express!

Ce tutoriel te permettra de réaliser un diagramme en bar interactif sur ton notebook, grâce aux codes suivanst et en 3 étapes seulement.

Etape 1 : Import librairies plotly express et pandas

import plotly.express as px
import pandas as pd

Etape 2 : Téléchargement des données

df_weather = pd.read_csv('https://raw.githubusercontent.com/murpi/wilddata/master/quests/weather2019.csv')

Etape 3 : Visualisation

Ecrivez le code suivant:
fig = px.bar(df_weather, x="DAY", y="MAX_TEMPERATURE_C", color="mois", hover_name='DATE',
animation_frame="MONTH", animation_group="DAY",title="Variations des températures max journalières en 2019", range_y=[0, 40])
fig.show()
